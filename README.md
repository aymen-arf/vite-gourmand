# Vite & Gourmand

Projet réalisé dans le cadre de l'ECF du TP Développeur Web et Web Mobile.

## Description

Application web de gestion et de commande de menus pour l'entreprise Vite & Gourmand.

## Technologies utilisées

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP 8
- MySQL
- Git / GitHub
- XAMPP

## Installation en local

1. Installer XAMPP.
2. Placer le dossier du projet dans `htdocs`.
3. Lancer Apache et MySQL depuis XAMPP.
4. Importer le fichier SQL dans phpMyAdmin.
5. Accéder au projet via `http://localhost/vite-gourmand/`.

## Fonctionnalités principales

- Consultation des menus
- Détail des menus
- Création de compte
- Connexion utilisateur
- Commande de menu
- Gestion utilisateur, employé et administrateur
